# Polkahub Deployer

This is Polkahub Deployer.

# Status

POC. Active development.

# Building

Install Rust:

```bash
curl https://sh.rustup.rs -sSf | sh
```

Build:

```bash
cargo build
```

# Run

```bash
cargo run
```
